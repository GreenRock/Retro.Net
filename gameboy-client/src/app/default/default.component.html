Dynamic Recompilation

What you're seeing is ASP.NET Core running Nintendo Gameboy software server side by dynamically recompiling Z80 opcodes into .NET expression trees.

Real Time

GPU rendering is completed on the server once per unique frame, compressed with LZ4 and streamed out to all connected clients over websockets.

Shared Controls

In between each unique frame, all connected clients can vote on what the next control input should be. The server will choose the one with the most votesâ€¦ most of the time.
